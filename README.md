# Medidor de rendimiento de combustible

Aplicación HTML para calcular rendimiento de combustible.

## Archivos

- `index.html`: aplicación principal.

## Publicación en GitHub Pages

1. Crear un repositorio en GitHub.
2. Subir el archivo `index.html`.
3. Ir a Settings > Pages.
4. En Branch elegir `main` y carpeta `/root`.
5. Guardar y esperar a que GitHub genere la liga.
